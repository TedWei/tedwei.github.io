javascript love with SICP

2015-07-04

最近在学习 SICP 。准备用 scheme 和 javascript 都实现一份书中的代码
今天贴两段关于计算阶乘的线性的递归和迭代

线性递归

scheme: 

(define (factorial n) 
      (if (= n 1)
           1
           (* n (factorial (- n 1)))))

javascript: 

var factorial = function(n){
                           if (n === 1) return 1;
                           return n * factorial(n-1);
                           }

线性迭代

scheme: 

(define (factorial n)
        (fact-iter 1 1 n))
(define (fact-iter product counter max-counter)
        (if (> counter max-counter)
            product
            (fact-iter (* counter product) (+ counter 1) max-counter)))

javascript: 

var factorial = function(n){
                           var value = 1;
                           for(var i = 1; i<=n ; i++){
                                value = value * i;
                           }
                           return value;
                           }
