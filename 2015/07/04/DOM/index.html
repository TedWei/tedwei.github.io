window.screen (MDN)

2015-07-04

原文链接为 https://developer.mozilla.org/en-US/docs/Web/API/Screen

window.screen
是跟设备界面相关的属性与方法

有以下属性

availHeight
可用的尺寸高度 单位为像素
会去除掉不可用的部分高度 例如菜单栏

availLeft
距离屏幕左侧的距离

availTop
距离屏幕顶部为
(MAC下浏览器全屏化之后 则为0 一般为23)

availWidth
可用的尺寸宽度

colorDepth
屏幕的色阶
一般有 8bit 15/16bit 18bit 24bit(true color) 30/36/48bit(deep color)

height
设备的屏幕高度 单位为px

orientation
屏幕的方向

pixelDepth
跟colorDepth一样

额外阅读：http://www.javascriptkit.com/howto/newtech3.shtml

width
设备的屏幕尺寸 单位为px

orientation
(safari不支持 其它最新的都支持) 移动端 只有最新的chrome和firfox支持
有三个属性
angle
onchange
type: portrait-primary, portrait-secondary, landscape-primary, landscape-secondary

moz有实现额外的两个属性
mozEnable
Boolean 类型 设置为false 可以关闭屏幕

mozBrightness
Controls the brightness of a device’s screen. A double between 0 and 1.0 is expected.
控制设备屏幕的亮度 返回值是浮点数 0到1.0之间

相关的属性有
window.screenX window.screenLeft
浏览器距离屏幕左侧的距离
window.screenY window.screenTop
浏览器距离屏幕顶端的距离
