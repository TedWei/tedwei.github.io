CSS3 动画相关

2015-07-16

CSS3 常用的跟动画相关的有三个属性 transition,transform,animation

transition的基本结构如下

transition : transition-property transition-duration transition-timing-function transition-delay

transition-property 指定某个属性改变时执行 transition 效果。主要值有 none,all,indent(属性名)。可指定的属性名可见 transition 可用属性List

transition-duration 是transition持续的时间 单位可为s 或者ms

transition-timing-function 指transition的变换速率。
主要值有
1.ease:（逐渐变慢）
2.linear:（匀速）
3.ease-in:（加速）
4.ease-out:（减速）
5.ease-in-out:（加速然后减速）
6.cubic-bezier:使用贝赛尔曲线)

transition-delay 改变属性值多少秒后执行transition效果

transform的基本结构如下

transform : none | rotate | scale | skew | translate | matrix;

transform-origin 定义transfrom的基点 支持水平方向的left,center,right 垂直方向的top,center,bottom

rotate : 2D旋转 值为负数则为逆向旋转 单位为deg
scale : 以倍数的缩放 scale(x,y) scaleX scaleY
skew: 扭曲角度 skew(deg,deg) skewX skewY
translate : 移动 translate(x,y) translateX translateY
matrix: 矩阵)

animation

animation: animation-name animation-duration animation-timing-function animation-delay animation-iteration-count animation-direction animation-play-state

animation-name : ‘leftToRight’
使用keyframes 定义的动画关键帧的属性变化

keyframes 'leftToRight' {
    0% {
        left: 0%;
    }
    33% {
        left: 33%;
    }
    67% {
        left: 67%;
    }
    100% {
        left:100%;
    }
}

animation-duration 定义动画持续时间
animation-timing-function 定义动画速率的变化 跟transition-timing-function 一样
animation-delay 定义动画延迟的时间
animation-iteration-count 动画循环的次数 可选number 和 infinite
animation-direction 值 normal alternate (偶次正方向播放 奇次反方向播放)
animation-play-state 值 running | paused
